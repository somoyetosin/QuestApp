# QuestApp

QuestApp is a simple application which generate questions randomly from a 
JSON file. It is mainly desinged with the Express Web Framework, PUG View
Engine as the user interface using the Bootstrap 4.0 for the design pattern.

This is a project from my tresshouse class of ExpressJS
